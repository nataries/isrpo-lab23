# Лабораторная работа №22. Интерактивные веб-приложения на JavaScript

## Описание проекта

В ходе лабораторной работы было создано интерактивное веб-приложение на JavaScript. В его основу легла идея создания калькулятора для простых вычислений.

## Функциональность

Интерактивный калькулятор поддерживает следующие базовые операции:

1. Сложение `+`
2. Вычитание `-`
3. Умножение `*`
4. Деление `/`

Реализована проверка на ошибки, и вывод соответствующих сообщений пользователю при их обнаружении. Например, ошибками считаются:

- подряд идущие знаки (операторы)
- оператор в конце строки
- ввод нескольких точек в одном числе
- выражение не может начинаться с оператора (за исключением отрицательного числа с оператором `-`)

Также есть обработка ошибок для следующих ограничений:

- проверка деления на ноль (вывод сообщения об ошибке)

Ошибки обрабатываются при помощи синтаксиса языка, а именно конструкции `try {} catch() {}`

```
calculate() {
        if (!this.expression || this.hasError) return;
        try {
            const result = this.safeEvaluate(this.expression);
            if (!isFinite(result)) {
                throw new Error("Некорректный результат!");
            }
            this.expression = String(result);
            this.hasError = false;
            this.updateDisplay();
        } catch (error) {
            this.showError("Ошибка");
        }
    }
```

## Архитектура

Класс `Calculator` содержит в себе код с реализацией логики работы калькулятора. Вынесен в отдельный класс, поскольку так повышается безопасность приложения и становится легко добавлять новые функции.

Структура файлов в папке `calculator`:

1. В файле `index.html` добавлены кнопки калькулятора, поле для ввода выражения и вывода результата
2. Код в `main.js` является черновым вариантом, где весь основной код по логике работы содержится в одном файле
3. `calculator.js` выносит логику работы калькулятора в отдельный файл с отдельными функциями
4. `style.css` содержит стили для оформления внешнего вида калькулятора

## Логика работы

**Последовательность действий при нажатии кнопок**

- Сначала необходимо ввести первое число, над которым вы хотите провести операцию (реализована возможность ввода отрицательного числа с оператором `-`)
- Потом нажимаете на кнопку того оператора, который необходим для вычисления
- Затем вы можете ввести второе число
- Если вы передумали или хотите вычислить другое выражение, нажмите кнопку очищения `C`
- Если хотите продолжить вычисление, нажмите на `=`, и в текстовом поле появится ответ выражения
  **Как работает валидация ввода?**

- Если при вводе в текстовое поле обнаружены подряд идущие знаки операторов или точки, калькулятор не позволяет ввод более одного
- При попытке деления на ноль в текстовом поле пишется `Ошибка`

**Как происходит вычисление?**

- Создание переменной `expression` , в которой хранится введённое выражение, а дальше будет использоваться для вывода результата
- Нажатие кнопок для ввода цифр в текстовое поле
- Нажатие на кнопки операторов калькулятора
- Проверка, нажата ли кнопка `=`: если да, тогда происходит вычисление и вывод накопленного выражения
